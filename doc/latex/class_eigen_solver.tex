\doxysubsection{Detailed Description}
This solver uses the Eigen Function Expansion method as detailed in ref 1 to calculate the electrode surface SOC. As such, it stores all the necessary variables required for the iterative calculations for all time steps. After initiating the class, the object instance can be called iteratively to solve for electrode surface SOC for all time steps. 

The equation solved is x\+\_\+j\+\_\+surf = x\+\_\+ini + (1/5)\texorpdfstring{$\ast$}{*}j\+\_\+scaled + 3 \texorpdfstring{$\ast$}{*} integration\{D\+\_\+sj \texorpdfstring{$\ast$}{*} j\+\_\+scaled / R\+\_\+j \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} 2\} \texorpdfstring{$\ast$}{*} dt + summation\{u\+\_\+jk -\/ 2 \texorpdfstring{$\ast$}{*} scaled\+\_\+j / lambda\+\_\+k \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} 2\} In the above equation, the integration is performed from t=0 to the current time. Moreover, the summation is performed from k=1 to k=N. Here k represents the kth term of the solution series.
