\doxysubsection{Detailed Description}
\begin{DoxyVerb}This class is used to create an electrode object. This object is used to store the electrode parameters and provide
relevant electrode methods. It takes input parameters from the csv file. The electrode parameters are stored as
attributes of the object.
\end{DoxyVerb}
