Detailed Description

This class contains the equations for the reduced order SEI growth model as mentioned in ref [1], with slight modifications.

  • Literature Reference:
    1. Randell et al. "Controls oriented reduced order modeling of solid-electrolyte interphase layer growth". 2012. Journal of Power Sources. Vol: 209. pgs: 282-288.
