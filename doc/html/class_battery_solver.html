Detailed Description

Contains the method to solve for the continuum-scale battery using the single particle model (SPM).

The model equations have been adateped from the following literature.

Guo, M., Sikha, G., & White, R. E. (2011). Single-Particle Model for a Lithium-Ion Cell: Thermal Behavior. Journal of The Electrochemical Society, 158(2), A122. https://doi.org/10.1149/1.3521314/XML

Solution BatterySolver::solve ( BaseCycler i_cycler,
int store_solution_iter = 1 )

Runs the iteration for the battery cell solver.

The solution time for the iteration is also printed in the terminal.

Parameters
i_cyclerThe instance of the cycler
store_solution_iteriteration at which at which to store the solution varaibles. For instance, the value of 5 means that the solution variables are stored after 5 iterations. Use the value of 1 to store solution variables obtained after all of the iterations. Note that the solutions variables from the first iteration is always stored.
